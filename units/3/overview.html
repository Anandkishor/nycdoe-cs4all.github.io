Introduction to Computational Media with p5.js

Unit 3 Media

Overview

In this unit students learn about working with media and asynchronicity. The preload function is used to load image, font and sound files before sketches are run, and callback functions are introduced. In the final project students combine images, fonts and sounds to create their own random meme generator.

Big Ideas

  • Digital media: the possibility combining and manipulating different kinds of media programmatically to create generative or interactive compositions
  • The possibility of creating custom media authoring tools

Enduring Understandings

  • Understand that sounds, images, and fonts can be combined and manipulated with code.
  • Understand the process of researching, editing, and manipulating media in order to create an interactive experience.

Goals

Understand asynchronicity
Use the preload function to ensure media is fully loaded before using it
Media
Incorporate images, fonts and sounds into a p5 sketch
Source, edit and combine media content

Learning Activities

1 . Loading media files: preload and callbacks

~15'
2 . Draw Images
Load and display image files.
~30'
3 . Load Fonts
Draw fonts, change their stroke and fill, draw within a box. Use custom fonts.
~30'
4 . Play sounds
Load and play sound files.
~30'

Final Project Prompt

Create a random meme generator: pick a collection of images and a collection of phrases. Each time you run your sketch, it should select one random image and one random text, creating a (potential) meme. What font are you using? What is your source for your images and text?

Level 1
Use loadImage and a custom font

Level 2
When the user clicks, change the color of the font, and the tint of the image

Level 3
Add sound: when the user clicks on the meme, have the doggos bark and the sneks hiss.
