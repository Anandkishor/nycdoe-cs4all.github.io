Big Ideas

  • Code as a creative and expressive medium
  • The concept of creating design systems: programmatically defining ranges for positions, dimensions, and colors, by using variables, randomness, and repetition. Drawing as collaboration between designer and system.

Enduring Understandings

  • Understand the basics of computation (execution flow, function calls, variables, loops)
  • Understand the basics of 2D computer graphics: drawing on a canvas, the coordinate system, calling drawing functions, setting colors in different color spaces
  • Understand the application of variables, repetition and randomness to the creation of parametric drawings

Unit Overview

In this unit students get acquainted with the p5 canvas coordinate system, drawing shapes and colors on it. Programming concepts covered include calling built-in functions with different parameters, working with variables (both built-in and custom), creating repetition (for loops and nested for loops for creating grids), and using controlled randomness.

Students are introduced to the p5 web editor and practice consulting the p5 online documentation. In the final project, students apply these skills to creating a design system for a music album cover.

Goals

1. Get familiarized with p5
Write and run sketches on the p5 online editor, place figures in the coordinate system, call functions with different sets of parameters, look them up in the p5 reference
2. Create a drawing using variables and randomness
Use variables and the random function to create a drawing that varies each time the sketch is run.
3. Understand color models
Use the RGB (Red, Green, Blue) and HSB (Hue, Saturation, Brightness) color systems. Define color ranges and palettes
4. Understand for loops
Create grids using nested for loops, manipulate it for creating compelling visual patterns

Learning Activities

1 . Create a grayscale drawing
Draw ellipses, rectangles, and other shapes. Change their fill and stroke to different shades of gray.
~45'
2 . Make it vary
Use built-in variables to set shape positions (mouseX, mouseY, width, height). Create new variables to determine position and shade. Set them to random values.
~45'
3 . Make it colorful
Create color compositions. Use the RGB and HSB color systems; add transparency.
~45'
4 . Repeat shapes, draw in grids
Use a for loop to draw the same shape many times, then draw each shape slightly differently. Use two for loops to draw a grid with rows and columns.
~45'

Final Project Prompt

Create a cover for a music album or a book that is generative. Each time you run your sketch it is different, and yet always within your design: it should have some degree of control and some degree of randomness.

Level 1
Use the p5 drawing functions and the random function.

Level 2
Use variables to set the position, size and color of your visual elements, making it easier to tweak your code as you iterate your design.

Level 3
Create repeating elements using for loops or nested for loops.
