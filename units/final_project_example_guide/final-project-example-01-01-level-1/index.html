Step by Step Guide: Interaction of Color, Level 1

To see the complete code and running sketch click here.

Description

This project is an abstract composition inspired by the book Interaction of Color by Josef Albers. Each time the sketch runs, a different shade is picked for the background and the foreground rectangle, which also changes its size, within a range: the result is different each time, but the composition retains a certain character.

Learning Activities Reference

This example project applies the following concepts from previous Learning Activities:

  • Drawing basic shapes (LA 1.1)
  • Setting stroke and fill; HSB color mode (LA 1.2)
  • Using the random function to set positions, dimensions, and colors (LA 1.2 and LA 1.3)

Steps Outline

  1. Set up the sketch
  2. Draw a random rectangle, within certain parameters

Step by Step

1) Set up the sketch

Within the setup() function, let's create a canvas, 600 pixels wide and 400 pixels high : createCanvas(600, 400);

Because we would like to vary the hue of our rectangle from red to green, we set the color mode to HSB: colorMode(HSB). We also turn off strokes: noStroke();

So far we have been drawing our rectangles from their top left corner. Now we use rectMode(CENTER) to draw them from the center (for more examples, see the p5 reference)

The last line in our setup function is noLoop(). This tells p5 to call the draw() function only once. We will look into this in detail in the next learning activity. If you are curious, comment this line out to see what happens.

Here is the setup code in full:

function setup() {
  createCanvas(600, 400);
  
  //Use the Hue, Saturation, Brightness color model
  colorMode(HSB);

  //Turn off drawing strokes
  noStroke();

  //Draw rectangles from their center 
  //(makes it easier to center it on the screen)
  rectMode(CENTER);

  //Only run the draw loop once. Try commenting this out and see what happens.
  noLoop();
  
}

2) Draw a random rectangle, within certain parameters

Our draw function is simple: we draw the background, set a fill color, and then draw a centered rectangle. Our visual composition will be different each time the program runs, similar, since it will vary within controlled ranges. To accomplish this, we use the random function.

We set the background to a shade of gray by setting its brightness to a random number between 0 and 100, and its saturation to 0 (no color). Given these two values, we can set the hue to be anything: the color will still be gray. 

background( 0, //hue. the background would be red if the saturation wasn't zero.
              0, //saturation. zero means no color (black, gray, white)
              random(100)); //brightness. this will be a shade of gray, between 0 (black), and 100 (white)

We then set the fill to a hue between red (0 degrees on the HSB color wheel) and green (90 degrees), and since we want the color to be vibrant and bright, independently of its hue, we set both saturation and brighness to be 100. 

fill( random(0, 90), //hue. a color between red (when 0), and green (when 90)
        100, //saturation. 100 means the color is vibrant.
        100);//brightness. 100 means maximum light.

Finally, we draw the rectangle. We center it by setting its position to (width/2, height/2). The width of our rectangle will be between a third (width/3) and two thirds of the canvas (width/3). Similarly, its height will be between a third and two thirds of the canvas' height. 

function draw() {
  rect(width/2, //x. width/2 indicates the horizontal center of the canvas.
      height/2, //y. height/2 indicates the vertical center of the canvas.
      random(width/3, 2*width/3), //width. this will be between a third of the width of the and two thirds.
      random(width/3, 2*width/3));//height. this will be between a third of the height of the canvas and two thirds.
}
