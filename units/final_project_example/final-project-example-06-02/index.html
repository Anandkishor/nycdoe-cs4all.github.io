Unit 6. Final Project Example, Level 2

Emoji Sea

Notes

Step by Step Guide

A sea of text-based emojis that jiggle around the canvas. In this Level 2 example, emojis are hand-drawn, photographed, and loaded into the sketch as images.

Loading too many large images will make your script run slower. For your reference: in this example, we are loading 24 PNG images. Each is 128 pixels wide x 116 pixels high, and about 8KB in file size.
