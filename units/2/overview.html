Introduction to Computational Media with p5.js

Unit 2 Response and Transformations

Overview

In this unit students learn to make their sketches respond to actions taken by the user, from moving the pointer to clicking or touching it over a certain area, to typing on the keyboard. The if statement is introduced. The map function is used to convert inputs into useful ranges.

The unit also covers applying transformations to the coordinate system ––translation, rotation, scaling––, and isolating them to affect only a portion of the code, using push() and pop().

For the final project, students create an expressive drawing tool.

Big Ideas

  • Interactive systems: programming visual systems that change over time and respond to user inputs.
  • Interaction design: defining the interactions between user and system.
  • Designing tools for creative expression.

Enduring Understandings

  • Understand that we can create our own drawing tools, and that the creation of tools can be expressive in itself.
  • Understand that the fast succession of still frames allows for the illusion of motion and for the system to respond to user clicks, hovers, and key presses.
  • Understand linear transformations of the coordinate plane (scale, translate, rotate), and of one single variable (map function)

Goals

1. Understand the flow of p5
How the draw loop makes it possible to create motion and interactivity
2. Respond to user actions
Key presses, mouse motion and clicks. Map values to useful ranges.
3. Transform shapes
Understand the linear transformations of the coordinate system (translation, rotation, scale); compose and isolate them; use them to create motion.

Learning Activities

1 . Draw with the mouse
Draw where the mouse is.
~45'
2 . Hover and click on things
When the user clicks, change the drawing color
~40'
3 . Press keys
When the user presses the spacebar, clear the screen (and other key-triggered behaviors)
~20'
4 . Map values
Map the mouse position values to hue and saturation.
~20'
5 . Translate, Rotate, Scale
Transform the entire drawing; transform some of its elements.
~40'

Final Project Prompt

Create a beautiful/whimsical drawing tool.

Level basic
Use the line function and the current and previous positions of the mouse to draw. When the space bar is pressed, clear the screen. When certain keys are pressed, change the drawing color, transparency, and/or stroke weight.

Level intermediate
Instead of the line function, draw with shape functions (triangle, rect, ellipse, bezier). Use transform functions to change the brush size, rotation, position relative to the mouse. These changes should be triggered by key presses, like the previous example.

Level advanced
Include a tool palette: buttons for picking different colors, size, stroke, rotation. Only draw when the mouse is pressed.
